# Prompt Generation

Prompts are generated by the remote API at runtime. The prompt templates are not stored locally.

When `/ab-test` runs, `matrix.py` sends your product data (name, selling points, audience, tone, discount code) to the prompt API, which returns finished prompts tailored to your product for each ad format and angle.

## Formats & Angles

| Format | Variant A | Variant B |
|--------|-----------|-----------|
| Podcast Ad (16:9) | Authority/Expert | Story/Transformation |
| UGC Ad (9:16) | Problem-Solution | Social Proof |
| Lifestyle Ad (9:16) | Morning Ritual | Active Lifestyle |
| Greenscreen TikTok (9:16) | Hook-first | Comparison |

## What the API Uses

- Product name and description
- Selling points (woven into dialogue)
- Discount code (added to CTA if present)
- Custom notes and CTA from brand profile
- AI context from asset analysis (product type, subject appearance, mood)
- Audio reference (if uploaded)
